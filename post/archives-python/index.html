🖋入门篇💻
注释
单行注释 以#开头
·#这是注释
多行注释 三个单引号或者双引号
'''
这是注释
'''
"""
这是注释
"""
变量
命名：包含字母、数字和下划线。不能以数字开头，不能用关键字和函数名。
赋值：例如 a = 3
链式赋值：链式赋值用于同一个对象赋值给多个变量
·>>>x=y=3等同于x=3,y=3
系列解包赋值：系列数据赋值给对应相同个数的变量。
·>>>a,b,c=1,2,3等同于a=1,b=2,c=3
可实现替换
·>>>a,b=2,4
·>>>a,b=b,a
·>>>printf(a,b)
·>>>4 2
删除变量：del
·>>>del a

数据类型
python中最基本的内置数据类型：
• Number（数字）
• String（字符串）
• List（列表）
• Tuple（元组）
• Set（集合）
• Dictionary（字典）
不可变数据（3 个）：Number（数字）、String（字符串）、Tuple（元组）；
可变数据（3 个）：List（列表）、Dictionary（字典）、Set（集合）。

数字：
整型int
浮点型float
布尔型bool
复数型complex

字符串：
字符串用单引号' 或者双引号"括起来。
·>>>str ='HelloPython'
字符截取格式
变量[头下标:尾下标]
